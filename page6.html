Song Creation


Going From Loop to Song

So far we have created a loop. However, we need more than this in order to have a complete song. Loops are great for getting ideas down or to create a section of a track, but we need to combine loops together with smooth transitions in order to make a successful song.

To start working towards a complete song, we will check out the playlist, which we access by pressing F5. This is where we can place our patterns that we create and have the computer play them one after the other. If you click inside of the playlist, a block that says Pattern 1 will appear there. This section functions similarly to the piano roll. You can put another copy of the pattern next to the first one.

If you press play, this will be nothing different than what we hear before. Now we need to create more loops or patterns to play in addition to the loop we already created. Press F4 to create a new pattern. Don't panic when you see the empty channel rack, that is saved under Pattern 1. This is Pattern 2. This is where you take what you have learned so far and create your own pattern with different drums or different chords and combine them together to make a complete song.

I know it feels like you have been left hanging, but this is where I will pass it off to the many great YouTube creators who have helped me learn everything I know about this process. There will be direct links to their channels on the next page.
